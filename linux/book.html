Comandos Basicos de linux

Dentro de linux hay bastantes comandos para hacer las mismas cosas aplicando diferentes parametros, lo que permite una flexibilidad increible a la hora de realizar desde una sencilla navegacion entre directorios hasta una busqueda compleja de permisos, aqui me voy a centrar en los comandos mas basicos de linux para la navegacion en el sistema operativo desde la consola de comandos.

A continuación se presentan algunos comandos básicos de Linux:

  • ls: Muestra el contenido del directorio actual.

  • cd: Cambia de directorio.

  • pwd: Muestra la ruta completa del directorio actual.

  • mkdir : Crea un nuevo directorio.

  • rm: Elimina un archivo.

  • rmdir: Elimina un directorio vacío.

  • cp: Copia archivos y directorios.

  • mv: Mueve archivos y directorios.

  • touch: Crea un nuevo archivo vacío.

  • cat: Muestra el contenido de un archivo.

  • head: Muestra las primeras líneas de un archivo.

  • tail: Muestra las últimas líneas de un archivo.

  • grep: Busca patrones en archivos.

  • chmod: Cambia los permisos de archivo.

  • sudo: Ejecuta un comando con privilegios de superusuario.

Estos son solo algunos comandos básicos de Linux. Hay muchos más comandos y opciones disponibles. Puedes explorar más sobre ellos y sus opciones utilizando el comando man seguido del nombre del comando (por ejemplo, man ls) para acceder al manual del comando y obtener más información.

A continuación se presentan los argumentos comunes para los comandos cat, grep, cp, mv, rm, ls, cd, pwd, mkdir, sudo, rmdir y chmod

Comando ls:

  • -l: Muestra una lista detallada del contenido del directorio.
  • -a: Muestra todos los archivos, incluyendo los ocultos.
  • -h: Muestra los tamaños de archivo en un formato legible para humanos.
  • -r: Muestra el contenido del directorio en orden inverso.
  • -t: Ordena el contenido del directorio por fecha y hora de modificación.

Comando cd:

  • cd: Cambia al directorio de inicio del usuario actual.
  • cd ruta: Cambia al directorio especificado por .
  • cd ..: Cambia al directorio padre.
  • cd -: Cambia al directorio anterior.

Comando pwd:

  • No tiene argumentos adicionales. Simplemente muestra la ruta completa del directorio actual.

Comando mkdir:

  • mkdir nombre_directorio: Crea un nuevo directorio con el nombre especificado.
  • mkdir -p ruta: Crea directorios recursivamente en la ruta especificada.

Comando sudo:

  • No tiene argumentos adicionales. Se utiliza para ejecutar un comando con privilegios de superusuario.
|

Comando cat:

  • cat archivo: Muestra el contenido del archivo en la salida estándar.
  • cat archivo1 archivo2: Concatena el contenido de los archivos y lo muestra en la salida estándar.
  • cat archivo1 archivo2 > archivo3: Concatena el contenido de los archivos archivo1 y archivo2 y lo redirige hacia el archivo archivo3.
  • cat archivo | grep patrón: Muestra las líneas del archivo que coinciden con el patrón especificado.

Comando grep:

  • grep patrón archivo: Busca el patrón especificado dentro del archivo y muestra las líneas que coinciden.
  • grep -r patrón directorio: Busca el patrón de manera recursiva en todos los archivos dentro del directorio y muestra las líneas que coinciden.
  • grep -i patrón archivo: Realiza una búsqueda sin distinción entre mayúsculas y minúsculas.

Comando cp:

  • cp archivo destino: Copia el archivo en la ubicación de destino.
  • cp archivo1 archivo2 directorio: Copia los archivos en el directorio especificado.
  • cp -r directorio1 directorio2: Copia el directorio y su contenido de manera recursiva.

Comando mv:

  • mv archivo destino: Mueve el archivo a la ubicación de destino.
  • mv archivo1 archivo2 directorio: Mueve los archivos al directorio especificado.
  • mv nombre_actual nombre_nuevo: Cambia el nombre del archivo o directorio.

Comando rm:

  • rm archivo: Elimina el archivo.
  • rm -r directorio: Elimina el directorio y su contenido de manera recursiva.
  • rm -f archivo: Elimina el archivo sin solicitar confirmación.

Comando rmdir:

  • rmdir directorio: Elimina el directorio vacío.
  • rmdir -p directorio: Elimina el directorio y sus directorios padres si están vacíos.

Comando chmod:

  • chmod permisos archivo: Cambia los permisos del archivo según los permisos especificados. Los permisos pueden ser especificados en notación octal (por ejemplo, 755) o en notación simbólica (por ejemplo, u+rwx, g+rx, o o-w).
  • chmod +x archivo: Otorga permisos de ejecución al archivo.
  • chmod -r directorio: Elimina los permisos de escritura y lectura en el directorio y su contenido de manera recursiva.
